# 3.15.2 Confluent release

3.15.2 has been released with the following changes:
<!-- more -->

## Ubuntu 26.04 node deployment.

Management node not supported (that will be a 4.0 thing), but nodes may be Ubuntu 26.04 now

## Routed deployment support for Ubuntu

Add routed deployment support for Ubuntu

## XCC3 improvements

Fix servicedata collection for older firmware and tolerate slower responses on firmware retrieval

## Container compatibility improvements

Improve compatibility with container environments without timedatectl available

## Python 3.12+ fix

Fix remoting from clients running python 3.12 or newer

## Collective issue fixed

Fix node group rename in a collective
