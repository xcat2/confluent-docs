# Stop preferring HWE kernel for Ubuntu

Ubuntu HWE has an issue with newer hardware, use the original kernel for now until their issues are addressed.

# Fixes and improvements for ESXi

Enhance install disk selection with fallback and more cleanly handle hostname in DHCP environments

# Track deployment target IP in `deployment.client_ip` attribute.

Some environments do not let confluent control addressing or even make the address stable, provide an attribute to let such environments track the IP address
that checks in when deployments check in.

# Fix discovery of select XCC2 firmware

Some error data has changed in certain XCC2 firmware that required specific handling.

# Fix kdump with diskless images

Diskless images would leave the imgutil build dracut configuration during boot, which broke kdump setup.  Fix this during boot

# Fallback to permissive behavior when a redfish attribute dependency has a broken attribute referenced

Certain redfish implementations had bugs that prevented settings, such mistakes now opt out of client-side setting checks

# Web GUI warns on browser blocked passkey registration

When the browser refuses passkey registration, feedback is now provided on why this is likely to be the case

# Affluent updates

The affluent switch agent has ported more of the functionality to the golang edition executed on modern Cumulus switches. Interface status, alert reporting, ntp configuration, and hardware event logging are now
implement in the golang executable.
