# Update affluent for broader switch compatibility

The new binary affluent was compiled against a newer x86_64 feature set than supported by some switches, this
has been relaxed

# Improved handling for aggregated nVidia components in redfish implementations

Recent nVidia implementations result in multiple redfish systems and managers being presented, provide
improved handling of this situation

# Improve nodeinventory handling of more components

Certain components would cause unexpected behavior in nodeinventory, this has been addressed

# Improve nodemedia upload

Recent changes to harden security resulted in reduced reliability for nodemedia upload. This has been addressed as well as preflight checks for common reasons upload would fail
